Langkah 1: Buat Proyek Supabase

Aplikasi ini membutuhkan database real-time gratis dari **Supabase** agar datanya aman dan bisa diakses dari HP anggota lain.

  1. Buka situs supabase.com dan buat akun/login.
  2. Klik tombol New Project.
  3. Beri nama proyek (misal: Bendahara KKN), tentukan password database Anda, pilih server terdekat (Singapore), lalu klik Create New Project.
  4. Tunggu sekitar 1-2 menit hingga status proyek siap.

💡 Jika sudah siap, klik tombol "Lanjut" di bawah.

Langkah 2: Buat Tabel Keuangan

Silakan salin skrip SQL di bawah ini, lalu jalankan di **SQL Editor** pada Supabase Dashboard Anda. 

create table transaksi (
  id uuid default gen_random_uuid() primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  tanggal date default current_date not null,
  tipe text not null check (tipe in ('pemasukan', 'pengeluaran')),
  kategori text not null,
  jumlah numeric not null check (jumlah > 0),
  keterangan text,
  petugas text default 'Bendahara'
);

alter table transaksi enable row level security;

create policy "Izinkan baca publik" on transaksi for select to anon using (true);
create policy "Izinkan CRUD publik dengan anon key" on transaksi for all to anon using (true) with check (true);
  1. Pada dashboard Supabase, klik menu SQL Editor di sidebar kiri.
  2. Klik tombol New query (tanda +).
  3. Tempelkan kode SQL di atas dan klik Run di bagian kanan bawah. Pastikan muncul tulisan Success.

Langkah 3: Masukkan API Credentials

Dapatkan API credentials Anda di menu Settings (ikon roda gigi) -> API pada dashboard Supabase. Tempelkan di kolom berikut.

* Informasi kredensial ini hanya akan disimpan secara lokal di browser Anda sendiri (Aman, tidak terkirim ke server mana pun).
